날씨 정보 API

지역 코드와 날짜 범위를 입력하면 해당 기간의 날씨 정보(최고/최저기온, 습도, 강수량, 날씨상태)를 반환합니다.

GET /api/v1/weather?api_key={key}®ion={코드}&start_date={YYYY-MM-DD}&end_date={YYYY-MM-DD}
파라미터 타입 필수 설명
api_key string 필수 발급받은 API Key
region string 필수 지역 코드 (11=서울, 21=부산, 22=대구, 23=인천, 24=광주, 25=대전, 39=제주)
start_date date 필수 조회 시작일 (YYYY-MM-DD)
end_date date 필수 조회 종료일 (YYYY-MM-DD)

API Key 발급

날씨 API를 사용하려면 API Key가 필요합니다.

교통 정보 API

지역 코드와 교통 유형(버스/지하철)을 입력하면 해당 지역의 노선 정보와 실시간 혼잡도를 반환합니다.

GET /api/v2/traffic?api_key={key}®ion={코드}&type=bus|subway
파라미터타입필수설명
api_keystring필수발급받은 API Key
regionstring필수지역 코드 (11=서울, 21=부산, 22=대구, 23=인천, 25=대전, 39=제주)
typestring필수교통 유형: bus / subway

API Key 발급

교통 API를 사용하려면 API Key가 필요합니다.

관광지 정보 API

지역 코드로 관광지 목록을 조회하거나, 관광지 ID로 위치·이미지·입장료·성수기/비성수기 등 상세 정보를 제공합니다.

GET /api/v3/tourist?api_key={key}®ion={코드}&category={카테고리}
파라미터타입필수설명
api_keystring필수발급받은 API Key
regionstring선택지역 코드 (미입력 시 전체 조회)
categorystring선택카테고리: 자연 / 문화 / 역사 / 테마파크

API Key 발급

관광지 API를 사용하려면 API Key가 필요합니다.
